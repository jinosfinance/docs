# Repository Structure

This document details the complete structure of the `jinosfinance/docs` repository.

```
jinosfinance/docs/
├── README.md                 # Repository root README
├── docs/                     # Documentation files
│   ├── README.md             # Main documentation index
│   ├── getting-started.md    # Getting started guide
│   ├── architecture.md       # System architecture overview
│   ├── api-reference.md      # Complete API reference
│   ├── security.md           # Security model and practices
│   └── faq.md                # Frequently asked questions
└── .git/                     # Git repository metadata
```

## File Descriptions

### Root Level
- **README.md**: Basic repository description and setup information

### docs/ Directory
- **README.md**: Main documentation landing page with index of all guides
- **getting-started.md**: Step-by-step guide for new users to sign up and use JinOS
- **architecture.md**: Technical overview of the JinOS system, tech stack, and monorepo structure
- **api-reference.md**: Complete REST API documentation with endpoints, request/response examples
- **security.md**: Detailed explanation of the non-custodial security model and Privy MPC
- **faq.md**: Frequently asked questions (currently empty, to be populated)
