## What is JinOS?
JinOS is a decentralized neobank agent built on BNB Smart Chain (BSC). It combines:
- **Self-custodial wallet** — Your keys, via Privy MPC. No one else can access your funds.
- **Virtual Visa cards** — Issue instant virtual cards funded directly from your on-chain wallet.
- **AI financial agent** — 24/7 automated portfolio management, yield optimization, and spend analysis.
- **Premium OS interface** — A glassmorphism dark desktop UI with real-time BSC data.
JinOS is non-custodial. The platform never holds your private keys.
